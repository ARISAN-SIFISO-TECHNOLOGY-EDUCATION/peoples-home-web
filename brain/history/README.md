# History

> The incident and evolution archive for The People's Home ecosystem.
> Records what happened — pivots, renames, retired ideas, technical incidents.
>
> **Protocol:** Append only. Never edit existing entries. Date every entry.
> The full history is in [`HISTORY.md`](./HISTORY.md).

---

## What counts as history

Record here when:
- Something was renamed (app, package, URL)
- An idea was retired or rejected (even after significant investment)
- A technical incident occurred (Play rejection, keystore rotation, build failure)
- A major pivot happened (web-first pivot, capability architecture reframe)
