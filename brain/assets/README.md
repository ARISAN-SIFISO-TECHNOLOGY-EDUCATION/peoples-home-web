## What belongs here

- Architecture diagrams (World A / World B flow, TPH Core dependency graph)
- UI wireframes for shared patterns (Passport flow, Household Mode, voice-first layout)
- Reference screenshots for competitive context
- Brand / logo references (do NOT store source files here — link to the source repo)

## What does NOT belong here

- App source assets (icons, splash screens) → live in the app repos
- Functional code → `brain/` is documentation only
- Large binary files → prefer referencing external storage

## Naming convention

```
YYYY-MM-DD-<description>.<ext>
```

Examples:
- `2026-06-27-world-a-world-b-architecture.png`
- `2026-06-27-tph-core-dependency.svg`
- `2026-06-27-passport-flow-wireframe.png`
