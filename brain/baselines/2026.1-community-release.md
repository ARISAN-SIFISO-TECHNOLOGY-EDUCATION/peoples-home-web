## 1. What this baseline marks

The point at which The People's Home stopped being *a set of apps being built* and became **an
institution with documented governance, architecture and release practice**.

Three things became true on this date:

1. The public site is **curated**, not exhaustive — six flagships, one per pillar.
2. Presentation and delivery each have a **written constitution** rather than scattered precedent.
3. The flagship app has a **platform seam**, so the web edition can innovate without ever putting
   the shipped Google Play release at risk.

## 4. Platform seam — Math Adventure RPG

One codebase, two editions. **No fork.** Established at `math-adventure-rpg@46dd5d9`.

```
src/platform/
  profiles.ts      PROFILE = play-store | community | development | demo
  platform.ts      PLATFORM = { isWeb, isMobile, isDesktop }
  flags/           per-profile feature registry (play-store = ALL_OFF)
  featureFlags.ts  isEnabled() — gated on profile AND web shell
  adapters/        stub interface (adopted in Phase C)

src/web/           story-theatre · living-world · animations · community · advanced-ui
                   (inert scaffolds — no gameplay logic yet)
```

| Profile | Edition | `base` | Experimental flags |
|---|---|---|---|
| `play-store` *(default)* | Stable / LTS — Google Play | `./` | all off |
| `community` | Community Edition — web | `/` | on |
| `development` | local | `/` | on |
| `demo` | showcase | `/` | on, minus kingdomMap + community |

Unknown or absent profile falls back to `play-store` — the safe default. `WEB_BUILD=1` still maps to
`community`, so existing Cloudflare Pages config keeps working.

**The Stable Build Principle:** the Play edition's gameplay, saves, progression, content and UX never
change without explicit founder approval. Architecture evolves *behind* the release, never through it.

**Deliberately deferred (Phase C):** core extraction — one subsystem at a time by stability
(utils/types → educational engines → gameplay → UI last), with a freeze between each.
*No file moves without a reason.*

## 7. What this baseline unlocks

Math Adventure's Community Edition can now grow — Story Theatre, Living World, advanced animation,
seasonal events — entirely behind feature flags, with the Play release untouched. And apps can be
promoted **Building → Now** without changing the site's architecture: only the `status` metadata moves.
